This library facilitates the publication of packages encompassing multiple workspaces as defined in a package.json file.

## Installation

```bash
npm install workspaces-publish --save
```

## Usage

```bash
npx workspaces-publish \
  --token <token> \
  --registry <registry> \
  --root <root>
```
